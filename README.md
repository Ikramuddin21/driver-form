## Overview

This project driver form submission & data showing.

## Technologies Used

- React.js
- Vite
- JavaScript
- Tailwind CSS
- React Router Dom

## Installation

Clone the repository:

```bash
git clone https://github.com/Ikramuddin21/driver-form
cd driver-form
npm install
npm run dev
```
